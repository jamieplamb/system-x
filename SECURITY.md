# Security

If you find a vulnerability, please don't open a public issue.

Use GitHub's private vulnerability reporting instead: the Security tab on this
repo, then "Report a vulnerability". That goes straight to the maintainer
without tipping anyone else off.

You'll get an acknowledgement within a few days and updates as the fix
progresses. Once a fix has shipped you're welcome to talk about the find, and
you'll be credited in the release notes unless you'd rather not be.

Supported versions: the latest 1.x release.
